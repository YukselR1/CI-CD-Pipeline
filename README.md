# 🛒 MegaMarket Backend Gateway

MegaMarket projesinin ödeme ve indirim süreçlerini yöneten, **CI/CD (Sürekli Entegrasyon)** odaklı geliştirilmiş profesyonel bir backend servisidir.

## 🚀 Proje Özellikleri
- **Flask API:** Ödeme ve indirim hesaplamaları için RESTful endpointler.
- **Unit Testing:** `pytest` ile %100'e yakın kod kapsamı (test coverage).
- **CI/CD Pipeline:** GitHub Actions ile otomatik test ve hata denetimi.
- **Validation:** Hatalı veri girişlerine karşı güçlü hata yakalama mekanizması.
